# attractiveness_penalties

----------------------------------------------------------------------------------

Title: Leveraging Genomic Data to Document Within-Race Attractiveness Penalties Among Black Americans

Journal: Sociological Science

Authors: Taddess, Zhang, and Trejo (Equal first authorship)

Date: May 2026

----------------------------------------------------------------------------------

This replication package contains code to reproduce the analysis, tables, and figures included in the main text and supplementary material. The Add Health survey and genetic data used in this project are restricted-use and cannot be redistributed with the replication code. Researchers must apply separately for access through Add Health/CPC and dbGaP.

----------------------------------------------------------------------------------

Description of the script files: 


1. 01-prepare_pheno.R: prepare survey data and merge with genetic similarity proportions
2. 02-analysis.R: main analyses and create tables and figures
